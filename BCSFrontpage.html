A Little About This Tutorial

This tutorial is aimed to teach you how to design and create your own experiment. We will be using general information about basic experimental design and how to use CrowdExperiment, software that has been created through the Human Language Processing (HLP) lab in order to run experiments over the web.

As you go through this tutorial you will also go through the self-paced reading paradigm explanation (SPR). The navigation bar is set up so that you can reference back any section that you need. SPR specific pages are in yellow and the general pages are in blue.

After each section of the tutorial, there is an optional quiz. We highly recommend you take these quizzes. They will test to make sure you understand what you read, but will also be able to guide you through the materials you do not understand.

For any items that are highlighted you can hover over them and text will appear.

After you have completed this tutorial, you will be ready to use what you have learned here and in class in order to create a replication study based on readings you have done.
